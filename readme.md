A very old-looking JRPG that is entirely based on a list. I'm currently trying to remake this to look better. It's the constraints of being in a list that make it graphically look poor, and it also was only the second of what I will call it a "real RPGS". First version was 2023.
However, I did update the enemies and allies to be more diverse in 2025.
Before that, enemies only did normal attacks, but now they can inflict status, disperse buffs, etc.
Hint: Save the pills and level up!
