# The crime of no return

A very old-looking JRPG that is entirely based on a text-based list on Scratch! It's the constraints of being in a list that make it graphically look poor, and it also was only the second of what I will call it a "real RPG". First version was 2023, in the midst of my examinations, because I was indeed bored! (Interesting choice in hindsight.)

This basically is a challenge of how to design a game with no base GUI elements, and only text-based! Like those MS-DOS games in the past. Luckily, this time, I had access to emojis.

However, I did update the enemies and allies to be more diverse in 2025.

Before that, enemies only did normal attacks, but now they can inflict status, disperse buffs, etc. Still more of a silly game than a serious one! I think still, few had made a RPG entirely based upon a text-only list.

Hint: Save the pills and level up! Beds recover HP!
